Dom

DOM eller document object model som det också heter handlar om några olika specifikationer av programmeringsgränssnitt för HTML- och XML-dokument. Det gör så att det ska göra det mycket lättare att ge nya möjligheter med att skriva webbtillämpningar vilket arbetar med till exempel XML-dokument. Javaprogram är ett exempel som hämtar data från ett XML-dokument.
